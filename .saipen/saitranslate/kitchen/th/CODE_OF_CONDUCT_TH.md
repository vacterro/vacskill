# พันธสัญญาผู้ร่วมสมทบ (Contributor Covenant Code of Conduct)

## คำมั่นสัญญาของเรา

ในฐานะสมาชิกผู้ร่วมสมทบผลงานและผู้นำของโครงการ เราขอให้คำมั่นว่าจะเป็นส่วนหนึ่งที่จะทำให้ชุมชนของเราปราศจากการล่วงละเมิดไม่ว่ารูปแบบใดก็ตามสำหรับทุกคน โดยไม่แบ่งแยกอายุ ขนาดร่างกาย ความพิการที่มองเห็นได้หรือมองไม่เห็น เชื้อชาติ เพศสภาพ การแสดงออกทางเพศ ระดับประสบการณ์ ระดับการศึกษา สถานะทางเศรษฐกิจและสังคม สัญชาติ รูปร่างหน้าตา เผ่าพันธุ์ ศาสนา หรือรสนิยมและการระบุตัวตนทางเพศ

เราขอให้คำมั่นว่าจะปฏิบัติและโต้ตอบด้วยวิธีที่เกื้อหนุนให้เกิดชุมชนที่เปิดกว้าง เป็นมิตร หลากหลาย ครอบคลุม และมีสุขภาพดี

## มาตรฐานของเรา

ตัวอย่างพฤติกรรมที่ส่งเสริมให้เกิดสภาพแวดล้อมเชิงบวก:
- แสดงความเห็นอกเห็นใจและความกรุณาต่อผู้อื่น
- ให้ความเคารพต่อความคิดเห็น มุมมอง และประสบการณ์ที่แตกต่าง
- การให้และการยอมรับคำติชมอย่างสร้างสรรค์
- รับผิดชอบและกล่าวคำขอโทษต่อผู้ที่ได้รับผลกระทบจากความผิดพลาด และเรียนรู้จากประสบการณ์นั้น
- ให้ความสำคัญกับสิ่งที่ดีที่สุดสำหรับชุมชนโดยรวม ไม่ใช่เพื่อตัวเราเองในฐานะปัจเจกบุคคล

ตัวอย่างพฤติกรรมที่ยอมรับไม่ได้:
- การใช้ภาษาหรือภาพสื่อในทางเพศ และการให้ความสนใจหรือการล่วงละเมิดทางเพศไม่ว่าในรูปแบบใดก็ตาม
- การก่อกวน (Trolling) การแสดงความคิดเห็นที่ดูถูกหรือเหยียดหยาม และการโจมตีส่วนบุคคลหรือทางการเมือง
- การคุกคามในที่สาธารณะหรือส่วนตัว
- การเผยแพร่ข้อมูลส่วนตัวของบุคคลอื่นโดยไม่ได้รับอนุญาตอย่างชัดเจน
- พฤติกรรมอื่นๆ ที่ถือได้ว่าไม่เหมาะสมในสภาพแวดล้อมของการทำงานอย่างมืออาชีพ

## ความรับผิดชอบในการบังคับใช้

ผู้ดูแลโครงการ (Project maintainers) มีหน้าที่ในการชี้แจงและบังคับใช้มาตรฐานเกี่ยวกับพฤติกรรมที่ยอมรับได้ของเรา และจะดำเนินการแก้ไขที่เหมาะสมและเป็นธรรมเมื่อพบเห็นพฤติกรรมที่พวกเขาเห็นว่าไม่เหมาะสม เป็นภัยคุกคาม ก้าวร้าว หรือเป็นอันตราย

## ขอบเขต

หลักจรรยาบรรณนี้มีผลบังคับใช้ภายในพื้นที่ทั้งหมดของชุมชน (เช่น issues, pull requests, discussions) และยังมีผลบังคับใช้เมื่อบุคคลใดบุคคลหนึ่งกำลังทำหน้าที่เป็นตัวแทนของชุมชนอย่างเป็นทางการในพื้นที่สาธารณะด้วย

## การบังคับใช้

กรณีที่มีพฤติกรรมการทารุณกรรม การคุกคาม หรือพฤติกรรมอื่นๆ ที่ไม่สามารถยอมรับได้ อาจรายงานถึงผู้ดูแลโครงการได้ผ่านทาง GitHub issue โดยทำเครื่องหมายว่า เป็นความลับ (confidential) หรือแจ้งผ่านรายละเอียดการติดต่อในโปรไฟล์ GitHub ของผู้ดูแล การร้องเรียนทั้งหมดจะได้รับการตรวจสอบและสอบสวนอย่างรวดเร็วและเป็นธรรม
