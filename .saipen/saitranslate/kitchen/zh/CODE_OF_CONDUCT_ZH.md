# 贡献者公约行为准则

## 我们的承诺

作为成员、贡献者和领导者，我们承诺让参与我们社区的每个人都享有没有骚扰的体验，无论年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同和表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、人种、宗教信仰，或性身份和性取向。

我们承诺以有助于建立一个开放、热情、多元、包容且健康的社区的方式行事和互动。

## 我们的标准

有助于营造积极环境的行为示例包括：
- 对他人表现出同理心和善意
- 尊重不同的意见、观点和经验
- 给予并优雅地接受建设性的反馈
- 承担责任，向受错误影响的人道歉，并从经验中学习
- 关注什么不仅对我们个人最好，而且对整个社区最好

不可接受的行为示例包括：
- 使用性化的语言或图像，以及任何形式的性关注或性暗示
- 恶意挑衅、侮辱或贬损性评论，以及人身攻击或政治攻击
- 公开或私下的骚扰
- 在未获得他人明确许可的情况下发布他人的私人信息
- 在专业环境中可被合理认为是不适当的其他行为

## 执行责任

项目维护者负责澄清和执行我们可接受行为的标准，并将对他们认为不适当、具威胁性、具冒犯性或有害的任何行为采取适当和公平的纠正措施。

## 范围

本行为准则适用于所有社区空间 (Issue、拉取请求、讨论)，也适用于个人在公共空间正式代表社区时。

## 执行

辱骂、骚扰或以其他方式不可接受的行为实例，可以通过标记为机密的 GitHub Issue 报告给维护者，或通过维护者 GitHub 个人资料上的联系信息报告。所有投诉都将得到迅速、公平的审查和调查。
